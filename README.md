# 7-8-anima-mui

Компонентная библиотека с анимациями, построенная на Material UI и TypeScript.

Описание проекта

Проект представляет собой набор UI-компонентов с анимацией, разработанных с использованием React + TypeScript и Material UI для быстрого прототипирования интерфейсов.
Использованы: TypeScript ~98 %, CSS, React, компоненты Material UI и набор анимаций.
